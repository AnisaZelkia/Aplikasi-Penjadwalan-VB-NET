# Aplikasi-Penjadwalan-VB-NET
Aplikasi menggunakan bahasa pemogramman Visual Basic, aplikasi mempunyai 2 user, yaitu:

# Mahasiswa dengan akses modul jadwal
# Admin dengan akses modul : 

        - Login
        - Data Pegawai
        - Dat aMatakuliah
        - Data Rungan
        - Penjadwalan
